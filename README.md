
# BOT Minecraft

ทำเพื่อการศึกษาเท่านั้น
## สิ่งที่ต้องโหลด

- [NodeJS](https://nodejs.org/en)

## Npm ใช้คำสั่งนี้
```bash
  cd botmcjs
  npm install
```
## Yarn ใช้คำสั่งนี้
```bash
  cd botmcjs
  yarn
```
  
## รันขึ้นเว็บ

```bash
node index.js
```
จากนั้นเข้า http://localhost:3000/
